Integrate SendFox Optin with your Zippy Course Plugin

== Description ==
Plugin allow you to integrate your Zippy Course Plugin with SendFox optin

**Usage**

*Basic usage*

Once you have installed and activated the plugin, navigate to Zippy Course, Settings, Email Lists. You will find Sendfox in list.
Enable sendfox and enter your access token and save changes. After that you will see SendFox in relevant all Courses and Product pages.

*IMPORTANT NOTES*

This plugin sends data to SendFox. You can find more details at https://sendfox.com/. Plugin is using API mentioned at https://sendfox.helpscoutdocs.com/article/135-endpoints

== Requirements ==
* PHP 5.3 or higher

== Translations ==
* None

== Installation ==
1. Upload the `zippy-sendfox-kit` folder to `/wp-content/plugins/`
2. Activate the plugin (Zippy Courses Sendfox Integration) through the 'Plugins' menu in WordPress
3. Use the functionality via Zippy Settings => Email Lists => SendFox

== Frequently Asked Questions ==

= Does plugin allow to send email via ZendFox? =

Yes Plugin will send all Zippy Courses pluin via SendFox if you have enable and placed right Access Tken.
